# gduthesis_bechelor v0.0.2

当前版本聚焦于 Word 导出链路的可用性修正，适合作为模板仓库的首个可分发预览版。

## 本版更新

- 补齐中英文摘要、目录、正文、参考文献、致谢、附录的基础导出流程
- 修正 Word 导出中的摘要标题、关键词、目录位置与目录样式
- 修正图题、表题、参考文献在 Word 中的主要样式问题
- 改进简单表格在 Word 导出中的保留策略，并补充三线表样式收口
- 补充 Windows 与类 Unix 一键导出脚本

## 使用建议

- PDF 主要用于格式核对，最终提交链路以 Word 为主
- 如果最终要交 Word，图片尽量使用 `PNG/JPG`，不要依赖 `PDF` 插图
- 复杂表格在 Word 中仍建议人工复核
- WPS 对目录字段支持不稳定，若目录显示异常，优先在 Microsoft Word 中更新目录

## 给已 clone 用户的升级方式

- 直接下载本 Release 附带的 `zip` 覆盖更新包
- 解压后覆盖仓库中的 `scripts/`、`tools/` 与导出脚本
- 若本地已有自行修改，建议先提交或备份后再覆盖
