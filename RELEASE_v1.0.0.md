# gduthesis_bechelor v1.0.0

首个公开整理版本，目标是提供一套可直接分发给同学使用的广东工业大学本科毕业设计（论文）模板，并附带可提交 Word 稿件的导出链路。

## 本版内容

- 提供可直接编译 PDF 的 LaTeX 模板
- 提供从 TeX 导出 `.docx` 的 Pandoc 流程
- 补齐中英文摘要、目录、正文、参考文献、致谢、附录的基础示例章节
- 修正摘要标题、关键词、目录、图题、表题、参考文献在 Word 导出中的主要格式问题
- 附录标题与附录图表编号在 Word 导出中按 `附录A / 附录B` 形式处理
- 新增 Windows 与类 Unix 一键脚本，降低同学直接使用门槛

## 使用建议

- PDF 主要用于格式核对，最终提交链路以 Word 为主
- 如果最终要交 Word，图片尽量使用 `PNG/JPG`，不要依赖 `PDF` 插图
- 复杂表格在 Word 中可能退化为仅保留表题，提交前需人工检查

## 给已 clone 用户的升级方式

- 直接下载本 Release 附带的 `.patch` 文件
- 在本地仓库执行 `git apply <patch-file>`
- 若本地已有自行修改，建议先提交或备份后再套 patch
